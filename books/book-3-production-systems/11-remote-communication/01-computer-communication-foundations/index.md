# Section: Computer Communication Foundations

**Purpose:** Understand how computers communicate over networks

This section covers the foundational concepts needed to understand how Android apps talk to remote servers.

## Concepts Included

- [x] [Why Networks Exist](./01-why-networks-exist.md) — The problem networks solve and why computers need to communicate
- [x] [Client-Server](./02-client-server.md) — The request-response model that underlies all web communication
- [x] [IP Address](./03-ip-address.md) — The numeric identifier that locates a device on a network
- [x] [Domain Name](./04-domain-name.md) — The human-readable address that maps to an IP address
- [x] [DNS](./05-dns.md) — The system that translates domain names into IP addresses
- [x] [Port](./06-port.md) — The numbered channel that identifies which service on a device to talk to
- [x] [Socket](./07-socket.md) — The endpoint of a network connection, combining IP and port
- [x] [Connection](./08-connection.md) — How two sockets establish a communication channel
