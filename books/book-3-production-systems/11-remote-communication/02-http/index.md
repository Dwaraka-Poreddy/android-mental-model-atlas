# Section: HTTP Protocol

**Purpose:** Understand how the HTTP protocol works

This section covers the HTTP protocol that powers web communication.

## Concepts Included

- [x] [HTTP](./01-http.md) — The protocol that powers data transfer on the web
- [x] [Request](./02-request.md) — The structure of an HTTP request: method, URL, headers, body
- [x] [Response](./03-response.md) — The structure of an HTTP response: status code, headers, body
- [x] [HTTP Methods](./04-http-methods.md) — The verbs (GET, POST, PUT, DELETE) that express the intent of a request
- [x] [GET](./05-get.md) — Fetching a resource without side effects
- [x] [POST](./06-post.md) — Creating a resource or submitting data
- [x] [PUT vs PATCH](./07-put-vs-patch.md) — The difference between replacing and partially updating a resource
- [x] [DELETE](./08-delete.md) — Removing a resource
- [x] [URL](./09-url.md) — The complete address that identifies a resource on the web
- [x] [Path Parameters](./10-path-parameters.md) — Variable segments in a URL that identify a specific resource
- [x] [Query Parameters](./11-query-parameters.md) — Key-value pairs appended to a URL to filter or modify a request
- [x] [Headers](./12-headers.md) — Metadata sent with a request or response (content type, auth, caching)
- [x] [Body](./13-body.md) — The payload of a request or response containing the actual data
- [x] [HTTP Status Codes](./14-http-status-codes.md) — The numeric codes that communicate the result of a request
- [x] [Status Code Categories](./15-status-code-categories.md) — The five categories: 1xx, 2xx, 3xx, 4xx, 5xx
- [x] [2xx Success](./16-2xx-success.md) — Codes that indicate a request succeeded (200 OK, 201 Created, 204 No Content)
- [x] [4xx Client Errors](./17-4xx-client-errors.md) — Codes that indicate the client made a mistake (400, 401, 403, 404, 422)
- [x] [5xx Server Errors](./18-5xx-server-errors.md) — Codes that indicate the server failed (500, 502, 503, 504)
- [x] [HTTPS](./19-https.md) — HTTP with TLS encryption, the standard for all production web communication
